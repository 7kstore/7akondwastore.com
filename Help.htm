  How to use the website

Work your way around the website by using the toolbar.

The 7AKONDWA STORE consists of products that you can purchase.

The ACTIVITIES area contains different activities to keep you busy while on the website. These activities include fun quizzes and games.

The SOCIAL page is where you can communicate to other users and the website administration.

The HELP page is where you are now. This page helps to make your access to this page easier and better.

7akondwa Store

The 7akondwa store is where you can find a range of different products you might like to purchase.

The first products that you find on the site are ones which are available in-store.

Each product will have a picture (if provided) displayed and beneath it will be the products name(if provided), the contact details of the products sellers and other relative information.

If an in store product has been recently bought, we will try as much and as fast as we can to let you know. Text can be displayed on the products description to alert customers that the product has been sold.

You can also find different material to stream and download on the site.

The 7akondwa merchandise page is where you can find products being sold by 7akondwa.These products can also be found on the 'in-store' area if they are in-store.

The services link provides various services which you may wish to acquire.The details of the service providers will be provided.

Activities

 The activities page is where you can find various activities to keep busy while on this website.

Quiz

For the quiz section, there are a number of quizzes which you can choose from depending on your interest.

There are different topics for the various quizzes that are on the site.

The first four answers (a,b,c and d) are related to the question and the fifth answer provides the option to choose if there are no correct answers provided or if the question or answers are impossible/not making sense.

NOTE: The original versions of the quiz games were fully automated but the current version uses microsoft powerpoint.

 

Crossword puzzle

For the crossword puzzle section, there are a number of crossword puzzles which you can choose to play.

There are several themes which you can choose from. For example Bible books and characters,football,music genres and others!

There are ten words which you have to find for each crossword puzzle.

NOTE: The crossword puzzles are not automated.

Go for it and have fun!

Brain teasers

The brain teaser activities offer a number of questions which are aimed at stretching your imagination. Having a partner or a group of friends to play with offers a better experience but it is not a must. Go on play and enjoy!

Code breaker

In this game a number of questions (3 or 4) will be asked. The answers to these questions will be used to create a code which if the answers are correct will manage to crack the code. Give it a go!

Social Site

The social page offers the opportunity to talk to others, catch up on whats going on online, watch videos and other highlights and ask questions or offer feedback if necessary.
